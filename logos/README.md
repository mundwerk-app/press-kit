# Mundwerk Logos

App-Icons in vier Größen. PNG mit Transparenz.

| Datei | Auflösung | Empfohlen für |
|-------|-----------|---------------|
| `mundwerk-icon-128.png` | 128×128 | Inline-Verwendung in Artikeln, Tabellen |
| `mundwerk-icon-256.png` | 256×256 | Standard-Press-Asset, Beitragsbild klein |
| `mundwerk-icon-512.png` | 512×512 | Headergrafiken, Hero-Bilder |
| `mundwerk-icon-1024.png` | 1024×1024 | Print, hochauflösende Hero-Sektionen |

## Wortmarke

Die geschriebene Wortmarke lautet **Mundwerk** (Markenname). Bei englischsprachigen Veröffentlichungen kann „Mundwerk" als Eigenname unverändert verwendet werden.

## Hintergründe

Das Icon ist auf transparentem Hintergrund. Auf hellen Hintergründen funktioniert es ohne weitere Bearbeitung; auf dunklen Hintergründen empfehlen wir die Verwendung mit Kontrast-Rahmen oder Original-Glyph als Vektor (auf Anfrage SVG verfügbar).

## Verwendungs-Hinweise

- **Erlaubt:** Verwendung in Reviews, Artikeln, Videos, Vergleichstabellen, Vorstellungs-Posts
- **Erlaubt:** Skalierung in beide Richtungen, solange Seitenverhältnis 1:1 bleibt
- **Nicht erlaubt:** Modifikation der Glyph-Form, Einbau in andere Logos, Verwendung als Teil von Werbung für unbeteiligte Produkte
